# Chocolate Sales Analysis Dashboard – Tableau

## Project Overview
Developed an interactive Tableau dashboard to analyze chocolate sales performance across countries, brands, and salespersons.

## Tools Used
- Tableau
- Excel
- Data Visualization
- Dashboard Design

## Dashboard Features
- Sales by Country Analysis
- Brand-wise Revenue Share
- Top 10 Salespersons
- Monthly Sales Trend Analysis
- Shipment KPI Monitoring
- Interactive Global Country Filter

## Key Insights
- Identified countries contributing highest sales revenue.
- Analyzed top-performing chocolate brands.
- Evaluated sales trends and seasonal patterns.
- Tracked top salespersons and shipment performance.

## Business Impact
Provided business insights to support sales optimization, regional strategy, and marketing decision-making.
