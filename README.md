# Chocolate Sales Analysis — Tableau

## Overview
Interactive Tableau dashboard analyzing chocolate product sales performance
across regions, products, and salespersons.

## Problem Statement
Sales teams need visibility into product performance, regional trends,
and individual salesperson productivity to drive revenue growth.

## Tools Used
- Tableau Desktop (filters, parameters, calculated fields, data blending)

## Key Findings
- Identified top-selling products and underperforming regions
- Analyzed monthly revenue trends and seasonal patterns
- Built salesperson productivity leaderboard

## How to View
Download the .twbx file and open in Tableau Desktop or Tableau Public.
